Lesson #6 - C.F.U - Conditonals Investigate

When creating an if-else-if statement you should always make your first condition the most specific. Write a short paragraph responding to the questions below. What does it mean to put the most specific case first? Why is it important to put the most specific case first? What types of errors does it help avoid?

To put the most specific case first means to put what will generally be in order. Example: Ratings for a movie would be placed from most amounts to fewer amounts. A G-rated movie should come first in the if-else order because it might restrict someone 13+ from viewing it. The computer program might read that the 13-year-old cannot watch the G-rated when they obviously can.It helps avoid errors such as not including the correct person/requirement for the person.

In your own words describe the benefits of creating functions in your code?

Functions allow you to store conditions in a single statement. These functions can be reused anywhere throughout the code and still do the same thing. It also helps tidy up your code to make it easier to debug and rewrite any code if necessary
