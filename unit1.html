A. The 8-bit system will be able to represent twice as many values as the 4-bit system.
B. The 8-bit system will be able to represent sixteen times as many values as the 4-bit system.
C. The 8-bit system will be able to represent eight times as many values as the 4-bit system.
D. The 8-bit system will be able to represent the same number of values as the 4-bit system, it will just include more zeros.

Answer choice B: The 8-bit system will be able to represent sixteen times as many values as the 4-bit system.

This is because an 8-bit system can represent 256 values, while a 4-bit system can only represnt 16 values.

Each time we add another bit, the amount of numbers we can make will double. For example, a 4-bit system can represent 16 values, but an 8-bit system can represent upto 256 values.

Lesson #9 - C.F.U - Lossless Compression

What is the most important quality of lossless compression?

A. The data is transformed to usually make it smaller. It can always be re-constructed back to the original.
B. The data is transformed to always make it smaller. It can always be re-constructed back to the original.
C. The data is transformed to usually make it smaller. It cannot always be re-constructed back to the original.
D. The data is transformed to always make it smaller. It cannot always be re-constructed back to the original.

Answer Choice A: The data is transformed to usually make it smaller. It can always be re-constructed back to the original.

This is true because the data transformed through lossless compression is usually smaller and can be reconstructed back to the original.

If you could talk to this author, how would you explain what is happening to their book?

The words in the book are not lost. The emailing service is just compressing the information to send it easily to the publisher. No data will be lost because it is was compressed through lossless compression. This type of compression will not delete any of the original data.
